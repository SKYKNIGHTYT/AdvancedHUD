# AdvancedHUD

AdvancedHUD is a mod that can be shaped and bent to anyone's will. Built on the Fabric modding platform, it is designed to be lightweight and resource efficient. While I have tried to minimise external dependencies, it requires Fabric API to work.

AdvancedHUD aims to eliminate the need of 3rd party libraries (aside from Fabric API) and also reduce resource usage by cutting down the extra code that is inherently copy-pasted between different individual mods with minor changes because they need them to work individually.

There are a lot of popular, customisable and somewhat resource efficient HUD mods out there, however they all lack one key detail, they are not unified and in some cases overlap, causing unexpected behaviour. Furthermore, most of these mods are created by different individuals or teams. Each one of them have their own way, preferences and design choices of making said mods, which may, make the user experience less clean overall. Another key detail to mention is that some of these mods require other library mods as (optional) dependencies, which adds unnecessary resource usage.

AdvancedHUD currently as of version 0.2.1 has the following features/can show the following:

FPS
Coordinates
Direction
Ping
Armor
The features can be turned on/off individually and removes the need of having a separate mod specifically for displaying armor.
